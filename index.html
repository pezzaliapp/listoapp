Come si usa

1. Listino

Carica un file Excel/CSV o un PDF. I dati restano nel browser (IndexedDB). Per i file tabellari, scegli quali colonne sono codice, descrizione e prezzo.

2. Preventivo

Cerca un articolo, premi Aggiungi, modifica quantità e sconto. Esporta il preventivo come PDF o copia il testo per inviarlo su WhatsApp/email.

3. Promo

La sezione mostra in automatico le promo attive (data di validità nel range corrente). Aggiornamento ogni 60 secondi mentre la pagina è aperta. Puoi anche importare promo locali da un file JSON o da un singolo allegato.

4. Installazione

  • iPhone (Safari): tocca Condividi → Aggiungi alla schermata Home.
  • Android (Chrome): menu → Aggiungi a schermata Home.
  • Desktop (Chrome/Edge): icona Installa nella barra degli indirizzi.

5. Privacy

Tutti i dati (listini, preventivi, promo locali) restano sul tuo dispositivo. Non vengono inviati a server.
